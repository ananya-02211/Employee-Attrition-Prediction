# Employee-Attrition-Prediction

## Objective

Predict whether an employee will leave the company using Machine Learning.

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Models Used

- Logistic Regression
- Random Forest
- Gradient Boosting

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results

The project identifies employees at risk of attrition and provides business recommendations to improve employee retention.
